SEEKER :  “Spiritual Healthcare” / การทรงเรียก / การทรงเลือก / การรับใช้พระเจ้า / การมองหาคริสตจักร พี่เลี้ยงและน้องเลี้ยง / การถวายสิบลด / ไฟแห่งพระวิญญาณ / การล้มในพระวิญญาณ การเจิม การเต็มล้นในพระวิญญาณบริสุทธิ์ / Burnout ทางฝ่ายวิญญาณ /หลักข้อเชื่อ 
การทรงเรียก (Calling)
การทรงเรียกไม่ใช่แค่ “ตำแหน่ง” แต่คือ “ตัวตน”  พระเจ้าทรงเรียกเรา ก่อน ให้ทำงานรับใช้
** ตัวอย่างในพระคัมภีร์:
Jeremiah 1:5 — ถูกทรงเรียกก่อนเกิด
Ephesians 2:10 — ถูกสร้างมาเพื่อการดี
Romans 8:30 — ทรงเรียก ทรงชำระ ทรงยกชู
Key สำคัญ
การทรงเรียก = Identity ก่อน Activity  การทรงเรียกมักมาพร้อม “ความเจ็บปวด” ที่พระเจ้าใช้หล่อหลอม การรับใช้ (Ministry)
พระเยซูเป็นต้นแบบในการรับใช้ :
Mark 10:45 — มาเพื่อรับใช้
John 13 — ล้างเท้าสาวก
**Burnout ทางฝ่ายวิญญาณ  รับใช้จนไม่มีเวลาเงียบกับพระเจ้า  ความคาดหวังจากคน
การมองหาคริสตจักร (Church Searching)  ควรมองหา 6 ด้านนี้:
1. พระวจนะถูกต้องหรือไม่  สอนพระคัมภีร์ครบถ้วน  ไม่บิดเบือนเพื่อเอาใจคน
2. วัฒนธรรมรักและปลอดภัย มีความโปร่งใส  มีระบบดูแล
3. ผู้นำมีชีวิตที่สอดคล้องคำสอนและศาสนศาสตร์
4. มีระบบพี่เลี้ยง (Discipleship)
5. มีพื้นที่ให้เติบโต ไม่ใช่แค่ทำงาน
6. มีการดูแลสุขภาพจิต (สำคัญมาก)
การหาพี่เลี้ยงด้านจิตวิญญาณและหลักการให้คำปรึกษาด้านจิตเวช / สุขภาพจิต ตามหลักจิตบำบัด
**ต้องการ:
Spiritual director
Christian counselor
Mentor ที่เข้าใจทั้งพระคัมภีร์และจิตวิทยา
หลักข้อเชื่อ การแบ่งคณะ / นิกายของคริสเตียน (ภาพรวมใหญ่)
คริสเตียนทั่วโลกแบ่งออกเป็น 3 กลุ่มหลัก
กลุ่มดั้งเดิม (Historic / Liturgical)
 1.1 คาทอลิก (Catholic)
 1.2 ออร์โธดอกซ์ (Orthodox)  เช่น กรีก ออร์โธดอกซ์ / รัสเซีย ออร์โธดอกซ์
 โปรเตสแตนต์ (Protestant)   ความรอดโดยพระคุณ (Sola Gratia)
 โปรเตสแตนต์แตกออกเป็นหลายสาย
2.1 สายหลัก (Mainline Protestant) เช่น:  ลูเธอรัน เมธอดิสต์ เพรสไบทีเรียน แองกลิกัน
2.2 สายอีแวนเจลิคัล (Evangelical)  การบังเกิดใหม่ การประกาศข่าวประเสริฐ  การศึกษาพระคัมภีร์จริงจัง พบมากในคริสตจักรไทยปัจจุบัน
2.3 สายเพนเทคอสต์ / คาริสแมติก (Pentecostal / Charismatic)
เริ่มต้นปี 1906 ที่เหตุการณ์ Azusa Street Revival  การทรงงานของพระวิญญาณบริสุทธิ์  การอธิษฐานเผื่อ ของประทานฝ่ายวิญญาณ  การรักษาโรค  บรรยากาศนมัสการค่อนข้างเข้มข้น
***คริสตจักรอิสระ (Non-denominational)
ไม่สังกัดนิกายใหญ่
โครงสร้างยืดหยุ่น
เน้นความสัมพันธ์มากกว่าระบบ
พบมากในเมืองใหญ่ และในองค์กรคริสเตียนรุ่นใหม่
