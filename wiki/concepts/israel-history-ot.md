# ประวัติศาสตร์อิสราเอล (OT)

เส้นเรื่องประวัติศาสตร์ในพระคัมภีร์เดิม — "ประวัติศาสตร์จากมุมมองของพระเจ้า" ที่มีพระคริสต์เป็นศูนย์กลาง (confirmed — [[sources/israel-history-ep1]], [[sources/israel-history-ep2]], [[sources/bible-survey-ot]])

## Timeline (กคศ. = ก่อนคริสตศักราช, conservative dating)
- **~2166** ทรงสร้าง / น้ำท่วมโลก / อับราฮัม
- **2166-1876** ยุคบรรพบุรุษ (อับราฮัม → ครอบครัวยาโคบอพยพไปอียิปต์)
- **1876-1446** อิสราเอลอยู่อียิปต์
- **1446-1406** อพยพ + ถิ่นทุรกันดาร (โมเสส, บัญญัติสิบประการ)
- **1406-1043** ยึดคานาอัน (โยชูวา) + ยุคผู้วินิจฉัย
- **1043-931** ราชาธิปไตยอาณาจักรเดียว (ซาอูล, ดาวิด, ซาโลมอน)
- **931-722** แบ่งเหนือ-ใต้
- **722-586** อาณาจักรเหนือล่ม (อัสซีเรีย) / ยูดาห์ (ใต้) ดำรงต่อ
- **586-538** เยรูซาเล็มถูกทำลาย เชลยบาบิโลน 70 ปี
- **538-430** กลับแผ่นดิน (เอสรา-เนหะมีย์) ภายใต้เปอร์เซีย/กรีก/โรมัน

## องค์ประกอบสำคัญ
- **3 รูปแบบการปกครอง**: โดยพระเจ้า (Theocracy) → ราชาธิปไตย → กอบกู้หลังเชลย
- **คำเรียกชนชาติ**: ฮีบรู → อิสราเอล → ยิว/ยูดาย
- **อาณาจักรเมโสโปเตเมีย**: สุเมเรียน → อัคคาเดียน → บาบิโลน → อัสซีเรีย → นีโอบาบิโลน → เปอร์เซีย
- **เส้นทางอพยพ** 18 จุด (ราเมเสส → ทะเลแดง → ซีนาย → คาเดช → จอร์แดน → เยรีโค)
- **ผู้วินิจฉัย**: โอทนีเอล, เอฮูด, เดโบราห์, กิเดโอน, เยฟธาห์, แซมซัน
