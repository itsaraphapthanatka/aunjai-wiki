# SEEKER หลักศาสนศาสตร์

> ที่มา: `raw/bible/SEEKER หลักศาสนศาสตร์ (1).md` — กรอบ "Spiritual Healthcare" ด้านการทรงเรียก การรับใช้ การดูแลฝ่ายวิญญาณ + ภาพรวมนิกาย

กรอบเชิง **อภิบาล/สาวกภาพ** (pastoral + discipleship) ผสมหลักจิตวิทยา/สุขภาพจิต — ใกล้เคียงบทบาทของน้องอุ่นใจในการดูแลคู่สนทนา

## Key claims
- **การทรงเรียก (Calling)** = Identity ก่อน Activity — พระเจ้าเรียกที่ "ตัวตน" ไม่ใช่แค่ "ตำแหน่ง" (เยเรมีย์ 1:5, เอเฟซัส 2:10, โรม 8:30) มักมาพร้อมความเจ็บปวดที่ใช้หล่อหลอม
- **การรับใช้ (Ministry)**: พระเยซูเป็นต้นแบบ (มาระโก 10:45, ยอห์น 13 ล้างเท้า)
- **Burnout ฝ่ายวิญญาณ**: รับใช้จนไม่มีเวลาเงียบกับพระเจ้า + ความคาดหวังจากคน
- **การมองหาคริสตจักร — 6 ด้าน**: พระวจนะถูกต้อง · วัฒนธรรมรัก+ปลอดภัย · ผู้นำมีชีวิตสอดคล้องคำสอน · มีระบบพี่เลี้ยง (discipleship) · มีพื้นที่เติบโต · **มีการดูแลสุขภาพจิต**
- **ที่ปรึกษาฝ่ายวิญญาณ**: ต้องการ spiritual director / Christian counselor / mentor ที่เข้าใจทั้งพระคัมภีร์และจิตวิทยา
- **ภาพรวมนิกาย** (ดู [[concepts/christian-denominations]]): ดั้งเดิม (คาทอลิก/ออร์โธดอกซ์) · โปรเตสแตนต์ (Mainline / Evangelical / Pentecostal-Charismatic) · คริสตจักรอิสระ (Non-denominational)

## Notable quotes
> "การทรงเรียก = Identity ก่อน Activity"

## Gaps
- เป็นโครงร่าง (outline) — แต่ละหัวข้อสั้น ขาดรายละเอียดหลักการให้คำปรึกษาเชิงลึก
- เชื่อมจิตวิทยา+ศาสนศาสตร์ แต่ไม่ระบุกรอบจิตบำบัดเฉพาะ
