# น้องอุ่นใจ — System Overview

**Raw source:** `raw/aunjai-system-overview.md`

## Key Claims
- น้องอุ่นใจ คือ LINE OA chatbot + AI companion ของ FEBC Thailand สำหรับงานดูแลจิตใจและ discipleship
- Admin URL: https://openclaw.appreview.cloud/admin/
- เป้าหมาย: 50,000 users ภายใน 12 เดือน
- ผู้ใช้ทั้งหมด ~10,048 คน (ณ 2026-06-05)

## Funnel Stages
Journey ของผู้ใช้ผ่าน 4 ขั้น:
1. **Engage** — พบเจอ content ครั้งแรก
2. **Connect** — interact สม่ำเสมอ
3. **Transform** — discipleship เชิงลึก
4. **Send** — พร้อมรับใช้ / เข้าคริสตจักร

## Circle Levels
- C1 = casual follower
- C2 = engaged learner
- C3 = committed disciple
